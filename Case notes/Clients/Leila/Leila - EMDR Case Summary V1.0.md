# Leila — EMDR Case Summary V1.0
## Client background
- Leila — Iranian female, mid-30s.
- Initially presented for stop smoking hypnosis; treatment successful.
- Subsequently disclosed:
- Acrimonious divorce — loss of marriage, identity disruption.
- Acute stress: mother and younger sister currently in Tehran during active bombing (Israel/US conflict).
- High anxiety, intrusive imagery, helplessness, anticipatory grief.
## Presenting issues
- Anxiety and hypervigilance.
- Intrusive imagery (news, phone, separation).
- Helplessness and anticipatory grief.
- Ongoing external threat; not a closed historical trauma.
## Formulation
- Dual stressor:
- Relational trauma (divorce).
- Ongoing traumatic threat + potential bereavement (family in war zone).
- Emerging theme: loss of safety, loss of attachment figures, loss of future.
### Session 1 — Online EMDR (recent event focus)
### Target
- Image: phone screen with missed calls from mother.
- Negative cognition (NC): "I am powerless".
- Positive cognition (PC): "I can cope and stay present".
- SUDS: 8/10.
- VOC: 2/7.
- Body: chest tightness, shallow breathing.
### Processing (brief sets; ~10–12 sets)
- Set 1: SUDS 8→7; image of phone; panic spike.
- Set 2: "I can't reach them"; throat tight.
- Set 3: no image; body anxiety only.
- Set 4: flash to argument with ex-husband.
- Set 5: "Everything is falling apart".
- Set 6: image of Tehran street from news footage.
- Set 7: tearful; SUDS 6.
- Set 8: blank; dissociative moment, re-grounded.
- Set 9: "I should be there".
- Set 10: SUDS 5; breathing slows slightly.
- Set 11: image returns; less vivid.
- Set 12: SUDS 5; session contained.
### End session 1
- Not fully processed.
- Client stabilised.
- SUDS ~5.
- VOC remains 2–3.
### Session 2 — Online EMDR (expanding network)
### Target
- Same network (phone/mother) with divorce links emerging.
- NC: "I am losing everything".
- PC: "I can survive this".
- SUDS: 7/10.
- VOC: 2/7.
- Body: chest + stomach heaviness.
### Processing (~12–15 sets; non-linear)
- Set 1: SUDS 7→6; phone image.
- Set 2: jump to divorce argument.
- Set 3: "He left me when I needed him".
- Set 4: image of empty house.
- Set 5: no image; heaviness in body.
- Set 6: memory of mother cooking; grief spike.
- Set 7: SUDS 6→5; crying.
- Set 8: news image overlay; bombs/sirens.
- Set 9: "I can't protect them".
- Set 10: brief sense of distance from image.
- Set 11: SUDS 5→4; sadness without panic.
- Set 12: image of sitting alone in London flat.
- Set 13: "I am on my own".
- Set 14: body softening slightly.
- Set 15: SUDS 4; session closed.
## Body scan
- Residual tension present (chest + stomach).
- No full resolution.
## Outcome
- SUDS reduced but not cleared.
- VOC increased slightly (2→4 range).
- Client more able to tolerate affect.
- Processing remains active.
## Clinical frame
- Work opening into double bereavement process:
- Loss of marriage (attachment rupture).
- Threatened/anticipated loss of family + homeland safety.
- Target network expanding appropriately.
- No premature closure.
## Supervision questions
- How best to pace EMDR when trauma is ongoing (not past)?
- When to remain in recent event vs move to earlier attachment targets?
- How to manage dual-track grief (divorce + potential bereavement) within EMDR?
